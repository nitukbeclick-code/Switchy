איך לחסוך בגלישה — ניהול נתוני סלולר

סלולר· 20 ביוני 2026· 4 דק׳ קריאה
בקצרה: הגיגה נגמרים? פתרון מהיר: הפעילו Low Data Mode (iPhone) / Data Saver (Android), עצרו הורדות אוטומטיות ב-Spotify/Netflix/YouTube, ובדקו מי שורף ב-Settings → Cellular. לטווח ארוך — בדקו אם המסלול שלכם מתאים לצריכה האמיתית שלכם.

כמה גיגה צריכים — בדקו צריכה אמיתית

  • iPhone: הגדרות → סלולרי → גלול למטה → רשימת אפליקציות עם צריכה. אפסו ב-Reset Statistics בתחילת חודש.
  • Android: הגדרות → רשת → שימוש בנתונים → בדקו כל אפליקציה.
  • ממוצע ישראלי: 8–15GB לחודש. עם נסיעות/Waze/YouTube — 15–30GB. נטפליקס בסלולר = ~3GB לשעה.
  • אם צורכים פחות מ-8GB — מסלול ₪30–₹40 עם 20GB מספיק. אם 20GB+ — שקלו שדרוג.

הגדרות חיסכון — iPhone

  • Low Data Mode: הגדרות → Wi-Fi → (!) → Low Data Mode. ואז: הגדרות → סלולרי → נתוני סלולר → Low Data Mode. מפחית רקע אוטומטי.
  • Wi-Fi Assist: הגדרות → סלולרי → Wi-Fi Assist — כבו! זה עובר לסלולר כש-WiFi חלש ושורף ג'יגות.
  • Background App Refresh: הגדרות → כללי → רענון אפליקציה ברקע → ה-GFE. כבו לאפליקציות שלא צריכות עדכון רציף.
  • iCloud Backup: הגדרות → [שם שלכם] → iCloud → Backup → כבו 'Backup Over Cellular'.
  • App Store: הגדרות → App Store → Automatic Downloads — כבו עדכונים אוטומטיים על סלולר.

הגדרות חיסכון — Android

  • Data Saver: הגדרות → רשת → שימוש בנתונים → Data Saver → הפעל. מגביל רקע לכל האפליקציות.
  • בדקו אפליקציות 'כבדות': YouTube, Instagram, TikTok, Spotify — אפשרו להם גלישה רק ב-WiFi.
  • Play Store: הגדרות → הגדרות → Preference → עדכן אפליקציות → רק ב-WiFi.
  • WhatsApp: הגדרות → Storage → Media download → בסלולר: כבו הורדות אוטומטיות.
  • Google Photos: הגדרות → Backup → Cellular data backup → כבו.

אפליקציות שאוכלות הכי הרבה גיגה

  • YouTube: ~1GB לשעה ב-720p, ~3GB ב-1080p. הגדרו איכות ל-480p בסלולר: הגדרות YouTube → Data Saving → Limit mobile data usage.
  • Spotify/Apple Music: הורדות אוטומטיות של פלייליסטים ברקע. כבו הורדה בסלולר.
  • Instagram/TikTok: מזינים וידאו 24/7. הגבילו נתוני סלולר בהגדרות המכשיר.
  • iCloud/Google Photos: סנכרון תמונות בסלולר. הגבילו ל-WiFi בלבד.
  • Waze/Google Maps: ~50–100MB לנסיעה ארוכה. הורידו מפות אופליין מראש ב-WiFi.

WiFi במקום סלולר — מקסמו שימוש

  • הצטרפו ל-WiFi בכל מקום אפשרי: בית, עבודה, קניון, מסעדות. WiFi = חינם לנתונים.
  • WiFi Calling: שיחות טלפון דרך WiFi — גם אם הקליטה 0 סורגים, השיחה עובדת. פחות סלולר.
  • Hotspot: אם יש לכם גיגות עודפות ולבן זוג/ילד נגמרו — שתפו Hotspot.
  • WiFi ציבורי: קניות/קפה/ספריות — בדרך כלל מאובטח מספיק לגלישה רגילה (לא ב-banking).

מתי כדאי לשדרג מסלול במקום לנהל

  • אם חורגים חודש אחר חודש ומשלמים extra: שדרוג מסלול לרוב זול יותר מחריגות.
  • המסלול הבא גדול מהצריכה ב-30%+ — שדרג. המסלול הנוכחי גדול מהצריכה ב-50% — ירד מסלול.
  • חשב: 3GB ב-₪5/GB חריגה = ₪15 לחודש. מסלול גדול יותר = ₪10/חודש יותר. שדרוג משתלם.

שאלות ותשובות

אפליקציות עדיפות ב-WiFi — אבל WiFi Assist עובד 'בשבילי' — למה לכבות?

WiFi Assist של Apple אמור לעזור, אבל בפועל לרוב עובר לסלולר כשה-WiFi איטי — גם אם ה-WiFi מספיק לרוב הפעולות. מומלץ לכבות ולחסוך שרוף לא מודע.

כיצד לבדוק כמה גיגה נשארו?

iPhone: הגדרות → סלולרי → ב-'Current Period' תראו צריכה (אפסו בכל ראש חודש). Android: הגדרות → רשת → Data usage. ספקים ישראלים: אפליקציית הספק מציגה יתרה בזמן אמת.

האם הורדה מוקדמת (Offline) חוסכת גיגה?

כן — Netflix/Spotify/Google Maps עם תוכן מורד מראש ב-WiFi חוסכים גיגות משמעותיות בנסיעה. הורידו ב-WiFi בלילה לפני הנסיעה.

כמה גיגה WhatsApp לוקח?

הודעות טקסט — כמעט כלום. תמונות: ~0.5–1MB כל אחת. וידיאו: 10–30MB לסרטון. שיחות קוליות: ~1MB לדקה. וידיאו-קול: ~5MB לדקה. הגדירו הורדה אוטומטית ל-WiFi בלבד.
