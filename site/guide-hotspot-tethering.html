Hotspot — שיתוף אינטרנט מהטלפון

סלולר· 21 ביוני 2026· 3 דק׳ קריאה
בקצרה: Hotspot = שיתוף גלישת טלפון למחשב/טאבלט/מכשירים אחרים. עובד בישראל ברוב המסלולים. צריכת גיגות: גלישה 1GB/שעה, YouTube HD 2–3GB/שעה, Zoom 1GB/שעה. כלול ברוב מסלולי הגיגות — אין תוספת עלות.

בעמוד הזה

  1. כיצד מפעילים Hotspot?
  2. האם Hotspot כלול במסלול שלי?
  3. כמה גיגות Hotspot צורך?
  4. טיפים לחיסכון בגיגות Hotspot

כיצד מפעילים Hotspot?

  • iPhone: הגדרות → כלל הנתונים האישיים → אפשר נתונים אישיים. בחרו: WiFi / Bluetooth / USB.
  • Android (Samsung): הגדרות → חיבורים → Mobile Hotspot and Tethering → Mobile Hotspot.
  • USB Tethering: חיבור ישיר USB ממחשב לטלפון = מהיר יותר מ-WiFi Hotspot.
  • Bluetooth Tethering: איטי יותר. מתאים לטאבלט קרוב.
  • טיפ: הגדירו סיסמה חזקה ל-Hotspot WiFi — מניעת שכנים שמשתמשים ללא ידיעתכם.

האם Hotspot כלול במסלול שלי?

  • ברוב המסלולים הישראלים: Hotspot כלול ללא תוספת עלות.
  • FUP ל-Hotspot: חלק ממסלולי 'ללא הגבלה' כוללים FUP מחמיר על Hotspot (מהירות תרד מוקדם יותר).
  • בדיקה: קראו תנאי המסלול — חפשו 'שיתוף גלישה', 'Hotspot', 'תצורת נתונים אישיים'.
  • ספקים שמתירים: גולן, HOT Mobile, פרטנר, פלאפון, סלקום — כולם.
  • מסלולים עסקיים: לרוב Hotspot ללא הגבלה. בדקו תנאים.
  • רוצים מסלול עם הרבה גלישה ל-Hotspot? השוו ב-/cellular.html וב-/plans.html.

FUP — הסעיף הקטן שמאט אתכם

במסלולי 'ללא הגבלה' רבים יש FUP נסתר ל-Hotspot: אחרי כמות מסוימת המהירות מואטת. קראו את תנאי המסלול וחפשו 'שיתוף גלישה' לפני שאתם סומכים עליו לעבודה.

כמה גיגות Hotspot צורך?

  • גלישה רגילה (Google, Gmail, WhatsApp): ~0.5–1GB/שעה.
  • YouTube HD (1080p): ~2.5GB/שעה. SD (480p): ~0.7GB/שעה.
  • Zoom (720p): ~0.9GB/שעה. 1080p: ~1.5GB/שעה.
  • עבודה מהענן (Google Docs, Office 365): ~0.3GB/שעה.
  • Netflix: HD ~3GB/שעה. SD ~0.7GB/שעה. 4K ~7GB/שעה.

טיפים לחיסכון בגיגות Hotspot

  • הגבילו מכשירים: Hotspot WiFi → הגדלות → מספר מכשירים מחובר. הגבילו ל-1–2.
  • YouTube: הורידו מראש (YouTube Premium) או הגדירו 480p כברירת מחדל.
  • Windows Update: Windows מוריד עדכונים ברקע. סמנו Hotspot כ-'Metered Connection' (הגדרות → רשת → Properties → Metered).
  • iCloud Sync: ב-Mac — כבו סנכרון iCloud בזמן Hotspot.
  • Background Apps: סגרו אפליקציות ברקע שצורכות נתונים (Dropbox, Google Photos).
  • בחו״ל ה-Hotspot צורך מהחבילה — תכננו גודל eSIM בהתאם. ראו /guides/guide-esim-vs-roaming.html.

Hotspot בחו״ל = לתכנן ג'יגות

אם תשתפו אינטרנט למחשב בחו״ל, הצריכה מזנקת. בחרו חבילת eSIM גדולה יותר (15GB+) והשוו אפשרויות ב-/abroad.html.

שאלות ותשובות

האם Hotspot מאיץ את הסוללה?

כן — Hotspot = WiFi פעיל + GPS (לרוב) + מסך. הטלפון מתחמם ומתרוקן מהר. חברו לחשמל בשימוש ממושך. כיבוי GPS + מסך = מאריך סוללה.

כמה מכשירים ניתן לחבר ל-Hotspot?

iPhone: עד 5 מכשירים. Android: לרוב 5–8. בפועל: 2–3 מכשירים = ביצועים טובים. יותר מ-5 = האטה ניכרת.

האם Hotspot מאובטח?

WPA2/WPA3 (WiFi חזק) + סיסמה = בטוח מספיק לשימוש יומיומי. USB Tethering = הכי מאובטח (חיבור ישיר, ללא WiFi). Bluetooth = מאובטח אך איטי.

Hotspot vs. ראוטר 4G — מה ההבדל?

Hotspot מהטלפון: נוח, ללא ציוד נוסף, סוללה מתרוקנת. ראוטר 4G נפרד (₪150–₪300): סוללה עצמית, אינטנסיבי לחיבורים רבים, נוח יותר כ-FWA זמני. לשימוש קצר → Hotspot מהטלפון. לצוות עבודה → ראוטר 4G.
